### Usage (code for Animate CC)
```javascript
gotoFrame(this.mc, 120);
```
<br />


### Config
| Variable | Type | Description |
|-------------|:-------------:|-------------|
| mc | Object | - |
| end_frame | Number | - |
| tween | Number | `true` by default, but if set `false`, each call of function would move timeline by one frame |
